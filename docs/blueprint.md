# **App Name**: Cake Tree Delights

## Core Features:

- Hero Section with Parallax: Visually engaging hero section with a large heading, subheading, soft blurred background with café ambiance, parallax effect, and slow floating cake icons.
- Horizontal Scroll Ambiance Gallery: Horizontal scrolling image gallery with auto-scroll, manual drag scroll, rounded corners, soft shadows, 3D tilt hover effect, and smooth horizontal movement animation.
- Interactive Menu Section: Interactive menu with tab selection for categories (Cakes, Pastries, Snacks, Beverages, Specials), displaying image, title, description, and price for each item, with soft hover scale animation and slide/fade effects for category switching.
- Best Sellers/Specials Section: Highlight featured items (e.g., Red Velvet Cake, Chocolate Truffle) with full-width premium cards, dark overlay gradient, subtle glowing border, and slow fade + scale animation.
- Testimonials Section: Smooth auto-slider for customer reviews with customer photo, star rating, smooth slide transition, and soft fade on slide change.
- Google Map Integration: Embed Google Map location for 'The Cake Tree – Behind Jatra Hotel, Nashik' with smooth zoom-in animation and pin bounce effect.
- Contact Information Display: Display contact information including phone number, WhatsApp button, Google Maps link, address, opening hours, and Instagram button, with slide-up animation.

## Style Guidelines:

- Primary color: Soft golden tones (#F0E68C) to evoke luxury and warmth.
- Background color: Cream (#FAF9F6), almost white, for a clean and minimalist feel.
- Accent color: Chocolate Brown (#D2691E) for contrast and to emphasize bakery themes.
- Body and headline font: 'Poppins', a geometric sans-serif font with a precise, contemporary, fashionable feel.
- Note: currently only Google Fonts are supported.
- Use elegant, custom-designed cake and café-related icons.
- Maintain large and breathable spacing throughout the website to enhance the premium feel.
- Incorporate smooth, subtle animations and micro-interactions on hover to provide a highly interactive and premium user experience (0.4s animation duration).
